Le livrable est un projet JAVA APPLICATION il doit etre executer en utilisant l'IDE Netbeans ou Eclipse par exemple.

Etape 1: Executer le projet

Etape 2: Saisir les dimensions de la grille

Etape 3: Saisir la position initiale de l'aspriateur
NB: Il faut respecter les contraintes suivantes:
1-La position initiale ne doit pas depasser les dimensions de la grille.
2- L'orientation doit contenir un seul caractere(N,E,W,S).

Etape 3: Instructions
NB: Il faut respecter les contraintes suivantes:
1- les instructions doivent contenir seulement ces caracteres(D,G,A).
2-La position finale ne doit pas depasser les dimensions de la grille.

Le resultat va etre afficher sur l'output du projet.
